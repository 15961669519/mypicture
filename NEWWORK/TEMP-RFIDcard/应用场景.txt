### 一、校园相关场景
#### 1. 校园迎新寻宝闯关
- **布置方法**：
  - 卡片设计：制作“地点寻宝卡”，卡号=校园关键地点编号（如教务处=01、图书馆=02），印上学校LOGO；主机生成含7个随机编号的“任务卡”。
  - 隐藏位置：在教务处、图书馆门口、食堂、操场等50-100个关键地点，将寻宝卡藏在显眼且安全处（如门侧挂钩、公告栏角落），配套发放“编号-地点对照表”。
- **使用步骤**：
  1. 新生在迎新点领取任务卡和子机，用子机刷任务卡获取7个目标编号；
  2. 对照对照表找到对应地点，刷寻宝卡后再刷任务卡更新进度；
  3. 完成全部7个点位后，返回主机刷卡查看完成度和用时；
  4. 系统自动排名，前10名可领取迎新礼包。
- **宣传文案**：“新生必玩！解锁校园地图的趣味方式，闯关赢礼包，快速熟悉你的大学主场～”

#### 2. 校园安全应急知识闯关
- **布置方法**：
  - 卡片设计：“安全知识点卡”写入安全常识（如“遇火灾走疏散通道”），标注编号；任务卡按年级分类（低年级=基础知识点，高年级=进阶知识点）。
  - 隐藏位置：在教学楼疏散通道、实验室门口、消防栓旁等安全关键区域，将卡片藏在安全提示牌背面、课桌抽屉内（低年级），消防栓玻璃内侧贴磁吸卡。
- **使用步骤**：
  1. 班主任带领学生领取对应年级任务卡，讲解活动规则；
  2. 学生分组寻找知识点卡，用子机刷卡后复述知识点；
  3. 复述正确即可更新进度，完成7个点位集齐进度；
  4. 全班完成后，主机显示“安全小卫士”，开展知识抢答赛。
- **宣传文案**：“安全闯关大冒险！边玩边学应急技能，守护校园安全，争做安全小达人～”

### 二、教育培训场景
#### 3. 科学实验室安全打卡培训
- **布置方法**：
  - 卡片设计：“安全点寻宝卡”对应灭火器、洗眼器等安全设备，印上设备名称+注意事项；任务卡含7个安全点编号，配套“编号-安全点对照表”。
  - 隐藏位置：在实验室内各安全点旁（如灭火器顶部、洗眼器侧面）固定卡片，确保学生易发现且不影响设备使用。
- **使用步骤**：
  1. 学生领取任务卡、子机和对照表，明确7个目标安全点；
  2. 按编号找到对应安全设备，刷寻宝卡后再刷任务卡；
  3. 完成全部打卡后返回主机验证，确认无遗漏；
  4. 老师口头提问安全知识，考核通过即完成培训。
- **宣传文案**：“实验室安全不枯燥！打卡安全点位，牢记操作规范，解锁实验安全技能～”

#### 4. 课堂教学“闯关答题”系统（语文/数学/英语等）
- **布置方法**：
  - 卡片设计：“知识点卡”对应单道题目或知识点，卡旁贴印题目或二维码（扫码看题），标注题号；任务卡含7个随机题号，确保各组题目不同。
  - 隐藏位置：在教室不同角落（如讲台两侧、墙角书架）摆放知识点卡，按题号分区排列。
- **使用步骤**：
  1. 学生分组领取任务卡，用子机刷卡查看本组需完成的7个题号；
  2. 到教室对应区域找到知识点卡，完成题目作答；
  3. 刷知识点卡+任务卡，老师现场检查答案；
  4. 集齐7个正确作答记录，返回主机刷卡确认，获取积分或奖励。
- **宣传文案**：“上课也能玩闯关！趣味答题+寻宝，知识点记得牢，学习效率翻倍～”

#### 5. 儿童编程与电子制作课程Demo
- **布置方法**：
  - 卡片设计：“任务卡”含7个数字+完成状态存储字段，“寻宝卡”仅标注编号；硬件配套主机+子机，预留WiFi、OLED屏、蜂鸣器接口。
  - 隐藏位置：在实验室桌面分区摆放寻宝卡，按编号1-10随机分布，方便学生寻找测试。
- **使用步骤**：
  1. 学生分组分工（任务卡设计、子机读写、显示逻辑、WiFi同步）；
  2. 各组编写对应模块代码，搭建硬件连接；
  3. 测试阶段：用子机刷任务卡获取7个编号，寻找对应寻宝卡并刷卡更新状态；
  4. 验证WiFi同步功能，主机显示各组完成进度，讲解卡片存储原理。
- **宣传文案**：“编程+寻宝双解锁！亲手打造互动游戏，读懂RFID与WiFi同步的奥秘～”

#### 6. 非遗技艺沉浸式学习工坊
- **布置方法**：
  - 卡片设计：“技艺步骤卡”写入非遗关键流程（如剪纸“折-画-剪-展开”），配简短文字+编号；任务卡按“入门/进阶”分类，含7个步骤编号。
  - 隐藏位置：剪纸工坊藏于彩纸堆下、剪刀抽屉；陶艺工坊藏于陶泥块旁、工具架缝隙，贴防滑胶固定。
- **使用步骤**：
  1. 学员听老师讲解基础技艺，领取对应级别任务卡；
  2. 在工坊内寻找步骤卡，刷卡后按提示完成实操（如揉泥1分钟）；
  3. 实操达标后刷任务卡更新进度，解锁下一个步骤；
  4. 集齐7个步骤完成记录，领取技艺体验证书，带走个人作品。
- **宣传文案**：“非遗学习不无聊！步骤打卡+动手实操，解锁剪纸/陶艺技能，做传统文化传承人～”

### 三、商业/公共场馆场景
#### 7. 商场促销集章活动
- **布置方法**：
  - 卡片设计：“商户寻宝卡”标注商户名称+编号（1-100），任务卡含7个商户编号，印上“集卡抽奖”字样。
  - 隐藏位置：在商场各店铺收银台、展位显眼处摆放卡片，确保顾客消费/打卡时易获取。
- **使用步骤**：
  1. 顾客在服务台领取任务卡和子机（或持任务卡使用店铺固定子机）；
  2. 前往任务卡指定的7个商户，消费或打卡后刷商户寻宝卡；
  3. 刷任务卡记录进度，支持分次累计；
  4. 集齐7个点位后，到服务台刷卡抽奖，按完成时间分级发放奖品/优惠券。
- **宣传文案**：“逛街=寻宝+抽奖！打卡7家商户，解锁惊喜福利，手慢无～”

#### 8. 博物馆/展览导览解谜
- **布置方法**：
  - 卡片设计：“展品寻宝卡”贴于重要展项旁，标注编号；任务卡含7个随机编号，展览手册附“编号-展品对照表”。
  - 隐藏位置：在展项底座、说明牌侧面固定卡片，避免遮挡展品，确保参观者易发现。
- **使用步骤**：
  1. 参观者在入口领取任务卡、子机和展览手册；
  2. 刷任务卡获取7个编号，对照手册找到对应展品，阅读介绍；
  3. 在展品旁刷寻宝卡，再刷任务卡更新“已参观”状态；
  4. 集齐7个点位，返回入口主机刷卡，触发结尾语音，领取小礼品。
- **宣传文案**：“展览导览新玩法！按线索找展品，解锁趣味解说，完成探索赢好礼～”

#### 9. 酒店亲子住客互动礼遇
- **布置方法**：
  - 卡片设计：“福利卡”写入权益（免费早餐、儿童乐园畅玩），标注区域提示（客房区/公共区）；任务卡含7个福利编号，印上“终极礼遇”字样。
  - 隐藏位置：客房藏于枕头下、书桌抽屉；公共区藏于大堂绿植、餐厅餐桌垫下，确保住客偶然发现。
- **使用步骤**：
  1. 住客办理入住时领取任务卡，房间配备子机（或前台借用）；
  2. 入住期间寻找福利卡，刷卡验证后同步进度，支持分次收集；
  3. 退房前到前台刷任务卡核对进度，未集齐可兑换单项权益；
  4. 集齐7个点位，兑换终极礼遇（如家庭写真），福利卡循环重置。
- **宣传文案**：“住酒店送寻宝福利！客房+公共区藏惊喜，集齐解锁免费早餐+延迟退房，亲子出行更欢乐～”

#### 10. 宠物主题乐园互动闯关
- **布置方法**：
  - 卡片设计：“宠物关卡卡”写入互动任务（跨越障碍、听指令停留），标注“小型犬/猫咪”专属标识；任务卡含7个关卡编号。
  - 隐藏位置：草坪区藏于宠物隧道旁、障碍物下；互动区贴于玩具上、饮水池边缘，用防咬防水材质。
- **使用步骤**：
  1. 主人带宠物在入口领取任务卡，按品种/体型匹配关卡；
  2. 寻找对应关卡卡，刷卡后按提示完成互动任务（如跳过栏杆）；
  3. 任务完成后刷任务卡更新进度，子机显示“宠物小英雄+1”；
  4. 集齐7个关卡完成记录，兑换零食礼包和免费入园券。
- **宣传文案**：“宠物乐园新玩法！闯关互动+零食奖励，带毛孩子解锁趣味体验，还能赢免费入园资格～”

### 四、亲子互动场景
#### 11. 社区亲子定向越野/公园打卡
- **布置方法**：
  - 卡片设计：“点位寻宝卡”标注公园景观点（湖边、假山），任务卡含7个点位编号，配套“编号+线索”手册。
  - 隐藏位置：在公园景观点周边安全处摆放卡片，如凉亭座椅下、健身器材旁。

  1. 亲子家庭在活动起点领取任务卡、子机和手册；
  2. 按手册线索寻找7个点位，找到后刷寻宝卡+任务卡，子机提示音反馈；
  3. 支持中途休息，分次完成打卡；
  4. 返回起点刷卡，领取小奖品或“闯关成功”印章。
- **宣传文案**：“亲子周末去哪玩？公园定向越野+打卡寻宝，边玩边增进默契～”

#### 12. 亲子户外自然观察挑战
- **布置方法**：
  - 卡片设计：“物种卡”写入植物/昆虫名称+特征（如“蒲公英-白色绒球”），任务卡含7个季节性物种编号，标注“公园限定”。
  - 隐藏位置：在城市公园物种周边藏卡（蒲公英丛下、树干背面），用透明防水卡套保护。
- **使用步骤**：
  1. 家长带孩子在起点领取任务卡，写入当季7个物种编号；
  2. 按卡片特征描述寻找对应区域，找到物种卡后刷子机；
  3. 子机播报物种小知识，进度+1，继续寻找下一个；
  4. 集齐7个物种记录，返回起点兑换“自然观察小达人”电子证书（可打印）。
- **宣传文案**：“户外观察趣多多！寻找植物昆虫，学习自然知识，亲子共赴大自然的宝藏之旅～”

#### 13. 家庭宠物互动训练游戏
- **布置方法**：
  - 卡片设计：“指令卡”写入宠物指令（坐下、握手），编号1-20，用宠物偏爱颜色制作；任务卡含7个适合宠物体型的指令编号。
  - 隐藏位置：在室内宠物常活动区（沙发缝隙、宠物窝旁、玩具箱内）贴磁吸卡，卡套边缘圆润防划伤。
- **使用步骤**：
  1. 主人在主机处定制任务卡，选择“进阶模式”（按顺序匹配）；
  2. 用子机刷任务卡获取首个指令，引导宠物寻找对应指令卡；
  3. 找到后刷卡，让宠物完成指令（如叼玩具），子机提示“匹配成功”；
  4. 集齐7个指令完成记录，给宠物奖励零食，主机记录训练完成度。
- **宣传文案**：“宠物训练变游戏！寻宝+指令互动，增进主人默契，养出聪明乖毛孩～”

### 五、团建/聚会场景

### 八、公益/特殊群体场景

## 十、技术规格与实施指南

### 10.1 卡片数据结构
**寻宝卡：**
- 卡片类型标识：0x01
- 数字编号：1-100
- 校验码：CRC16

**任务卡：**
- 卡片类型标识：0x02
- 任务ID：唯一标识
- 目标数字数组：7个1-100数字
- 完成状态数组：7个布尔值
- 玩家信息：编号/姓名（可选）
- 时间戳：开始时间
- 校验码：CRC32

### 10.2 系统工作模式
**主机模式：**
1. 写卡模式：制作寻宝卡/任务卡
2. 任务模式：生成并写入任务
3. 读卡模式：读取卡片状态

**子机模式：**
1. 任务读取：获取玩家任务
2. 寻宝验证：刷卡匹配验证
3. 进度更新：更新任务状态

### 10.3 防错机制
1. **卡片校验：** 类型标识+校验码验证
2. **状态恢复：** 任务卡存储完整状态
3. **非法卡提示：** 蜂鸣器+LED提示
4. **数据备份：** WiFi同步云端备份（可选）

### 10.4 扩展功能接口
1. **显示接口：** I2C OLED接口
2. **音频接口：** PWM蜂鸣器/音频解码模块
3. **网络接口：** WiFi状态同步API
4. **存储接口：** SD卡数据日志（可选）
5. **电源管理：** 锂电池充放电管理

## 十三、常见问题解答

### Q1：卡片丢失或损坏怎么办？
**A：** 系统支持卡片重写功能，主机可重新写入相同数据到新卡。

### Q2：多人同时游戏会冲突吗？
**A：** 采用唯一任务ID设计，不同玩家任务卡数据独立，不会冲突。

### Q3：户外使用防水防摔吗？
**A：** 标准配置为室内使用，户外需定制防水防摔外壳（可选）。

### Q4：电池续航时间？
**A：** 基础配置续航8-10小时，扩展显示功能后续航5-6小时。

### Q5：能支持多少张卡片？
**A：** S50卡理论支持无限张，实际管理建议不超过500张。

### Q6：数据安全如何保障？
**A：** 采用校验码防篡改，重要数据可增加加密算法（可选）。
